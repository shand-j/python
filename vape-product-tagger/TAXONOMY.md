# Vape Product Taxonomy Reference

Complete reference guide for the comprehensive vaping product taxonomy used by the Vape Product Tagger.

## Overview

The taxonomy is organized into multiple dimensions to enable sophisticated filtering and navigation:

1. **Device Classification** - Type and form factor
2. **Flavor Taxonomy** - Hierarchical flavor categorization
3. **Nicotine Classification** - Strength and type
4. **Compliance Tags** - Age, regional, and safety tags
5. **Compatibility Tags** - Technical specifications

## Tag Application Logic

### Automatic Detection
The tagger automatically detects and applies tags based on:
1. **Keyword Matching**: Product titles and descriptions
2. **Pattern Recognition**: Nicotine values, device types
3. **AI Inference**: Complex flavor profiles, implicit characteristics

### Tag Hierarchy
Tags are organized hierarchically:
- Main category tags (e.g., "Fruit")
- Sub-category tags (e.g., "Tropical")
- Specific tags (e.g., "Mango")

All applicable levels are included for comprehensive filtering.

### Compound Flavors
Products with multiple flavors receive tags for all detected flavors:
- "Strawberry Banana" → Fruit, Berry, Strawberry, Tropical, Banana
- "Vanilla Tobacco" → Dessert, Custard, Vanilla, Tobacco, Classic

## Customization

### Adding New Keywords
Edit `modules/taxonomy.py` and add keywords to the relevant category:

```python
"Berry": {
    "tags": ["Berry", "Berry Flavor"],
    "keywords": ["strawberry", "raspberry", "blueberry", "YOUR_NEW_KEYWORD"]
}
```

### Adding New Categories
Add new sub-categories or main categories in the taxonomy structure:

```python
"NEW_FAMILY": {
    "tags": ["New Family", "New Flavor"],
    "keywords": ["keyword1", "keyword2"],
    "sub_categories": {
        "NEW_SUB": {
            "tags": ["Sub Category"],
            "keywords": ["sub_keyword"]
        }
    }
}
```

### Regional Variations
Adjust taxonomy for different markets by modifying keywords for local terminology.
